Title: Math Painting

Description: An app that allows users to give the start coordinates of geometrical
shapes such as rectangle and square, their dimension and their colors. The program
generates an image file containing the canvas on the shapes are drawn.

Objects:
Square: x
        y
        side
        color
        draw(canvas)

Rectangle:
        x
        y
        width
        height
        color
        draw(canvas

Canvas:
        width
        height
        color
        make(imagePath)
